# Employee Task & Productivity Tracker

This project is a Python-based command-line application designed to manage employee tasks and track productivity using a relational database.

## Features
- Add employee details
- Assign tasks with deadlines
- Update task status (Pending, In Progress, Completed)
- Store data using SQLite
- Generate a simple productivity summary report

## Technologies Used
- Python
- SQLite
- SQL queries (CRUD operations, JOIN, GROUP BY)

## Use Case
This project demonstrates basic task management and reporting functionality, similar to internal productivity tracking tools used in organizations.

## Future Enhancements
- Input validation
- Viewing task lists per employee
- Exporting reports to a file
